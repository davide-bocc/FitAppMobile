# FitAppMobile

FitAppMobile è un’app di supporto per Personal Trainer, pensata per semplificare la gestione degli allenamenti tra coach e allievi.  
*Questo è il mio progetto realizzato dopo aver seguito corsi di formazione su Coursera: "Python for Everybody" e "IBM Back-End Development", per mettere in pratica le conoscenze acquisite e sfidare le mie capacità con un progetto mobile complesso.*

## Descrizione del progetto

L’app permette a coach e allievi di interagire in modo semplice e veloce:

- **Registrazione** come coach o allievo.
- **Creazione schede allenamento** da parte del coach, caricate sul server con una singola chiamata.
- **Download scheda allenamento** da parte dell’allievo con una singola chiamata.
- **Esecuzione allenamento** con gestione di esercizi, tempi di recupero e note.
- **Salvataggio risultato** al termine dell’allenamento e caricamento sul server.

## Funzionalità principali

- Registrazione e login utente (coach o allievo)
- Creazione e gestione delle schede allenamento
- Sincronizzazione schede tra coach e allievo
- Monitoraggio e salvataggio dei risultati degli allenamenti
- Interfaccia mobile intuitiva

## Tecnologie utilizzate

- **Kotlin** 1.8.22
- **React Native** 0.74.7
- **Gradle** 8.9
- **JDK** 17
- **Firebase** per backend e storage
- **Hermes** abilitato
- New Architecture disabilitata

## Requisiti di sistema

- Android Studio o Xcode
- Emulator o dispositivo fisico Android/iOS
- Connessione Internet per sincronizzazione con Firebase

## Installazione

1. Clona il repository:
```bash
git clone https://github.com/davide-bocc/FitAppMobile.git
```

## Problemi noti

- Durante la generazione dell’APK, potrebbero verificarsi problemi legati all'autolink dei file nativi. Sto lavorando per risolverli.






# FitAppMobile

FitAppMobile is a support app for Personal Trainers, designed to simplify the management of workouts between coaches and trainees.  
*This project was created after completing Coursera courses: "Python for Everybody" and "IBM Back-End Development", to put into practice the knowledge I acquired and challenge my skills with a complex mobile project.*

## Project Description

The app allows coaches and trainees to interact in a simple and fast way:

- **Register** as a coach or trainee.
- **Create workout plans** as a coach, uploaded to the server with a single call.
- **Download workout plans** as a trainee, with a single server call.
- **Execute workouts** with exercise management, rest times, and notes.
- **Save results** at the end of the workout and upload them to the server.

## Key Features

- User registration and login (coach or trainee)
- Creation and management of workout plans
- Synchronization of plans between coach and trainee
- Tracking and saving workout results
- Intuitive mobile interface

## Technologies Used

- **Kotlin** 1.8.22
- **React Native** 0.74.7
- **Gradle** 8.9
- **JDK** 17
- **Firebase** for backend and storage
- **Hermes** enabled
- New Architecture disabled

## System Requirements

- Android Studio or Xcode
- Android/iOS emulator or physical device
- Internet connection for Firebase synchronization
